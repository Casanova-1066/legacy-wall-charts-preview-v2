# Cumulative Master Build

This ZIP is a complete replacement for the previous Dev2 Master. Do not merge older RC or Sprint ZIP files into it.

## Included in this build

- Create Account button beside Sign In on desktop and mobile.
- Dedicated Create Account page.
- Full name, confirm password, show/hide password, password strength and terms acceptance.
- Improved Neon Auth error and diagnostic flow.
- Premium knockout match cards with optional ET and PENS boxes.
- Knockout controls to show/hide ET and penalty boxes.
- Background fit modes: cover, contain, stretch and manual crop/zoom.
- Uploaded background pixel-dimension detection.
- Effective DPI estimate for the selected print size.
- Green/amber/red print quality guidance and 300 DPI warning.
- Editor zoom from 25% to 200%, fit-screen and 100% actions.
- Full-screen design inspection mode with front/back switching.
- Five World Cup 2026 style presets plus a custom blank builder.
- Expanded recent historical catalogue placeholders for Premier League, FA Cup and League Cup.
- Removed pnpm project declaration and private registry references.

## Workflow

1. Keep one folder named `Dev2 Master`.
2. Replace that folder with this ZIP's extracted contents.
3. Upload the complete project contents to the preview GitHub repository.
4. Test in Vercel Preview.
5. The next cumulative build must start from this master only.
